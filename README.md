- Browser

In order to run this project you need:

- Choose a directory
- Open a Terminal
- write: git clone git@github.com:zhorabay/Hello-world.git
- get into to directory "Hello-world"

### Setup

Clone this repository to your desired folder:

```sh

```

### Install
